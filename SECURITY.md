# Security Policy

Agent Loop Guard is a pre-1.0 educational open-source project. See the complete [threat model](docs/security.md) before relying on it with untrusted code or tools.

Do not report vulnerabilities with credentials, private traces, or exploit details in a public issue. Use GitHub private vulnerability reporting for this repository when it is available. Include the affected version, impact, reproduction steps, and a redacted proof of concept.

There is no guaranteed response SLA. Security reports are prioritized, but this project is not a certified sandbox, enterprise IAM product, or legal audit system.
